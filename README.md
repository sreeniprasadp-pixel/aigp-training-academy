# AIGP Training Academy

An interactive web application for AI Governance Professional (AIGP) certification preparation.

## Features

- 64 comprehensive lessons across 7 modules
- Interactive quizzes with detailed explanations
- Progress tracking with localStorage
- Mock exam simulator
- Trainer-guided learning experience

## Modules

1. **Artificial Intelligence Fundamentals** (18%)
2. **AI Impacts & Responsible AI Principles** (17%)
3. **Privacy Considerations** (16%)
4. **Regulatory Frameworks** (18%)
5. **Organizational AI Governance** (16%)
6. **AI Lifecycle Governance** (11%)
7. **Upskilling & Workforce Development** (4%)
